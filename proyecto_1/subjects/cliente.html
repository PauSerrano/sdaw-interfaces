Introducción

¿Qué aprenderás?

En este módulo aprenderás a comprender y aplicar los fundamentos esenciales del desarrollo web moderno. A través de ejemplos prácticos y actividades guiadas, descubrirás cómo funcionan los lenguajes, herramientas y metodologías que se utilizan hoy en día en entornos profesionales.

El objetivo es que adquieras una base sólida que te permita avanzar con seguridad hacia proyectos más complejos, entendiendo no solo el “cómo”, sino también el “por qué” de cada concepto.

Tema 1: Introducción a JavaScript

JavaScript es el lenguaje que permite añadir interactividad y dinamismo a las páginas web. En este primer tema conocerás sus conceptos fundamentales a través de explicaciones breves y ejemplos prácticos.

1. ¿Qué es JavaScript?

JavaScript es un lenguaje de programación que se ejecuta en el navegador y permite controlar el comportamiento de una página web. Gracias a él podemos responder a eventos, modificar el contenido y crear experiencias interactivas. 


  // Mostrar un mensaje en consola
  console.log("Bienvenido a JavaScript");

2. Variables: almacenar información

Las variables permiten guardar datos que luego podemos usar o modificar. En JavaScript moderno utilizamos let para valores cambiantes y const para valores fijos. 


  let nombre = "Pau";
  const curso = "JavaScript básico";

  console.log(nombre, curso);

3. Tipos de datos básicos

JavaScript trabaja con varios tipos fundamentales: números, textos, booleanos, listas y objetos. Estos tipos permiten representar información de forma flexible. 


  let edad = 20;                     // Number
  let activo = true;                 // Boolean
  let colores = ["rojo", "azul"];    // Array
  let usuario = { nombre: "Pau", rol: "alumno" }; // Object

Tema 2: Estructuras de control

Las estructuras de control permiten decidir qué instrucciones se ejecutan y en qué orden. Gracias a ellas podemos crear programas que reaccionan a condiciones, repiten acciones o recorren colecciones de datos.

1. Condicional

La estructura if permite ejecutar un bloque de código solo si se cumple una condición. Es la forma más básica de tomar decisiones en JavaScript. 


  let nota = 7;

  if (nota >= 5) {
    console.log("Has aprobado");
  }

2. Condicional

Cuando queremos ejecutar una acción si la condición se cumple y otra distinta si no se cumple, utilizamos if...else. 


  let edad = 16;

  if (edad >= 18) {
    console.log("Eres mayor de edad");
  } else {
    console.log("Eres menor de edad");
  }

3. Condicional múltiple

Cuando existen varias posibilidades, podemos encadenar condiciones usando else if. 


  let nota = 8;

  if (nota >= 9) {
    console.log("Sobresaliente");
  } else if (nota >= 7) {
    console.log("Notable");
  } else if (nota >= 5) {
    console.log("Aprobado");
  } else {
    console.log("Suspenso");
  }

4. Estructura

switch es útil cuando queremos comparar un mismo valor con múltiples casos. Hace el código más limpio que varios else if. 


  let dia = 3;

  switch (dia) {
    case 1:
      console.log("Lunes");
      break;
    case 2:
      console.log("Martes");
      break;
    case 3:
      console.log("Miércoles");
      break;
    default:
      console.log("Día no válido");
  }

5. Bucle

El bucle for permite repetir una acción un número determinado de veces. Es ideal para recorrer listas o ejecutar tareas repetitivas. 


for (let i = 1; i <= 5; i++) {
  console.log("Iteración:", i);
}
